Smart Surveillance with YOLOv8 and DeepSORT is a computer vision project for real-time people detection, tracking, and crowd monitoring. The system uses YOLOv8 to detect persons in webcam or video input, DeepSORT to maintain identities across frames, and frame-difference analysis to estimate motion in the scene. It displays current visible people, total unique tracked people, motion level, and a crowd alert when the number of people exceeds a configurable threshold. The project also saves an annotated output video and a structured log file for later analysis. Additional scripts are included to visualize people-count trends and optionally evaluate labeled predictions using standard metrics such as accuracy, precision, recall, and F1-score. This project demonstrates an end-to-end smart surveillance workflow using practical computer vision tools and is organized as an academic course submission with documentation, reusable scripts, and GitHub-ready presentation.
